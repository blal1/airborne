## 1. ACKNOWLEDGEMENTS & RESPONSES

| Phrase | Meaning | French Equivalent |
|--------|---------|-------------------|
| ACKNOWLEDGE | Let me know you received and understood | ACCUSEZ RÉCEPTION |
| AFFIRM / AFFIRMATIVE | Yes | AFFIRMATIF |
| NEGATIVE | No / Permission not granted / Not correct | NÉGATIF |
| ROGER | I have received all of your last transmission | REÇU |
| WILCO | I understand and will comply | COMPRIS / BIEN COMPRIS |
| UNABLE | Cannot comply with request/instruction | IMPOSSIBLE |
| STANDBY | Wait, I will call you | ATTENDEZ |
| GO AHEAD | Proceed with your message | PARLEZ |
| READ BACK | Repeat my message back to me | COLLATIONNEZ |
| SAY AGAIN | Repeat your message | RÉPÉTEZ |
| I SAY AGAIN | I repeat for clarity | JE RÉPÈTE |
| SPEAK SLOWER | Reduce your rate of speech | PARLEZ PLUS LENTEMENT |
| WORDS TWICE | Send every word twice | MOTS DEUX FOIS |
| CORRECTION | An error was made, correct version follows | CORRECTION |
| DISREGARD | Ignore | IGNOREZ |
| THAT IS CORRECT | Confirmation | C'EST EXACT |
| HOW DO YOU READ | What is readability of my transmission? | COMMENT RECEVEZ-VOUS ? |

## 10. READBACK (COLLATION)

These phrases are used when pilot reads back ATC instructions:

| Instruction Type | Readback Format |
|-----------------|-----------------|
| Taxi clearance | Runway, taxiway route, hold short |
| Takeoff clearance | "Cleared for takeoff runway XX" |
| Landing clearance | "Cleared to land runway XX" |
| Altitude assignment | "Climb/Descend to (level)" |
| Heading assignment | "Turn left/right heading XXX" |
| Frequency change | "Contact (facility) on (freq)" |
| Squawk code | "Squawk XXXX" |
| Hold short | "Hold short runway XX" |

## Key Differences ICAO vs FAA

| Situation | ICAO | FAA |
|-----------|------|-----|
| Ready for takeoff | "LINE UP AND WAIT" | "LINE UP AND WAIT" (was "Position and hold") |
| Wind report | "WIND 270 DEGREES 10 KNOTS" | "WIND 270 AT 10" |
| Altitude below FL180 | "ALTITUDE" | "ALTITUDE" |
| Altitude FL180+ | "FLIGHT LEVEL" | "FLIGHT LEVEL" |
